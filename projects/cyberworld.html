Overview

CyberSiege is an interactive game-based cybersecurity learning experience built in Minecraft Education Edition for students aged 11-14. The project uses a narrative-driven escape room format to teach core cybersecurity concepts such as encryption, brute-force attacks, phishing, and malware mitigation. All challenges were designed to align with the Curriculum for Wales (2022) and the Digital Competence Framework, combining technical problem-solving with engaging gameplay and storytelling. More is on my GitHub, as there are too many photos to show.

Key Features & Experience

  • Designed a game-based learning environment to teach cybersecurity concepts through interactive challenges
  • Applied practical knowledge of cybersecurity fundamentals, including encryption, brute-force attacks, phishing awareness, and malware mitigation
  • Built complex interactive systems using Minecraft Education tools (command blocks, redstone, NPCs, scoreboards)
  • Used Code Builder to automate world construction and improve development efficiency
  • Translated educational requirements into engaging playable mechanics aligned with curriculum outcomes
  • Implemented logic-based puzzles and conditional triggers to control player progression
  • Designed clear UX and accessibility features suitable for younger learners
  • Produced supporting educational materials, including a lesson plan and student workbook
