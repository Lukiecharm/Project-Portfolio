Overview

This project was done by myself and teammates, and involved developing an Android-based personal safety application designed to help students travel home safely at night through real-time monitoring and emergency response features. The app integrates GPS-based route mapping, motion sensor detection, and manual emergency triggers to proactively identify potentially dangerous situations. Built as a functional prototype under tight time and resource constraints, the system prioritised reliability, usability, and ethical considerations while meeting client requirements for platform, design, and emergency alerting.

Key Features & Experience

  • Developed a complete Android application using Android Studio with XML-based UI layouts
  • Applied Java/Kotlin for mobile app logic, adapting between languages and documentation limitations
  • Integrated GPS and third-party mapping services to support real-time route navigation
  • Implemented sensor-based motion detection using accelerometer data to identify abnormal events
  • Designed a user-controlled emergency alert system with countdown logic to reduce false positives
  • Evaluated ethical and safety considerations when designing systems involving emergency services
