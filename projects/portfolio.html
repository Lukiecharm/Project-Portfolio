Overview

This is the website you're currently viewing, my personal portfolio built entirely from scratch. It's designed to showcase my skills, projects, and personality in a clean, modern, and responsive way. The goal was to create something professional yet unique that stands out to recruiters and visitors.

Key Features

  • Fully responsive design, works perfectly on mobile, tablet, and desktop
  • Dark/light mode toggle with theme persistence (saved in localStorage) and helps with user preference
  • Smooth animations and hover effects throughout
  • Animated contact modal with working form (using FormSubmit.co AJAX)
  • Project cards with consistent layout and "View More" links to further see what the project is about
  • Clean, organised code split into multiple CSS and JS files
  • Minimal and fast-loading, no heavy frameworks
