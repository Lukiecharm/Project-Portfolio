Overview

This project involved designing and developing a smart IoT-based security camera system that enhances traditional surveillance by integrating computer vision and web technologies. Unlike conventional cameras that simply record footage, this system is capable of detecting objects and identifying the presence of people in real time with a percentage guess of how sure the system is of that object, with detection results displayed through a web interface. Built using Python, Raspberry Pi hardware, and a Flask backend, the project demonstrates how intelligent vision systems can be combined with IoT platforms to create more proactive and responsive security solutions.
